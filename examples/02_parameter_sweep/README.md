# 參數掃描範例

這個目錄包含單維度和多維度參數掃描的範例。

## 檔案

- `single_sweep.jl` — 單參數掃描
- `multi_sweep.jl` — 多參數掃描

## 執行

```bash
julia --project=. examples/02_parameter_sweep/single_sweep.jl
julia --project=. examples/02_parameter_sweep/multi_sweep.jl
```

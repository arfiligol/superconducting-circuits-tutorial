# sc preprocess hfss scattering

Convert HFSS exported S-parameter phase CSV into the standard dataset format (DB import) (formerly `sc-preprocess-phase`).

## Usage

```bash
uv run sc preprocess hfss scattering [OPTIONS] [csv ...]
```

## Arguments

| Argument | Description | Default |
|---|---|---|
| `csv` | HFSS phase CSV path(s) | - |

## Options

| Option | Description | Default |
|---|---|---|
| `--dataset-name` | Force specific Dataset Name (single file mode only) | |
| `--tags` | Comma-separated tags | |
| `--match` | Comma-separated keywords to filter files in directory mode | `Phase,S21,deg,rad` |

## Examples

**Basic import**
```bash
uv run sc preprocess hfss scattering data/raw/layout_simulation/phase/MyChip_Phase.csv
```

## Notes

!!! note "Database import"
    The current pipeline imports into SQLite; JSON output is deprecated.

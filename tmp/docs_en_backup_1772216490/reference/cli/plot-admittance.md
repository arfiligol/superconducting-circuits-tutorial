# sc plot admittance

Plot admittance ($Im(Y)$) line views to inspect resonance extraction quality.

## Usage

```bash
uv run sc plot admittance [OPTIONS] [datasets ...]
```

## Arguments

| Argument | Description | Default |
|---|---|---|
| `datasets` | Dataset names or IDs (multiple allowed) | required |

## Options

| Option | Description | Default |
|---|---|---|
| `--show-zeros` | Mark zero crossings ($Im(Y)=0$) | False |
| `--freq-min` | Min frequency (GHz) | - |
| `--freq-max` | Max frequency (GHz) | - |
| `--title` | Plot title | - |
| `--show` / `--no-show` | Open browser preview directly | `--show` |
| `--save-html` / `--no-save-html` | Save HTML output | `--no-save-html` |
| `--output`, `-o` | HTML output path | auto-generated |

## Examples

**Line view**
```bash
uv run sc plot admittance LJPAL658_v1
```

**Mark zero crossings and limit range**
```bash
uv run sc plot admittance --show-zeros --freq-min 4.0 --freq-max 8.0 LJPAL658_v1
```

## Notes

!!! note "CLI architecture"
    This feature is now under the `sc plot` command tree (no standalone `plot-admittance`).

# sc plot flux-dependence

Plot flux dependence scan data as heatmaps and slices.

## Usage

```bash
uv run sc plot flux-dependence [OPTIONS] [datasets ...]
```

## Arguments

| Argument | Description | Default |
|---|---|---|
| `datasets` | Dataset names or IDs (multiple allowed) | required |

## Options

| Option | Description | Default |
|---|---|---|
| `--parameter` | Parameter name (e.g. `S11`) | `S11` |
| `--view` | View (`amplitude`, `phase`, `combined`, `all`) | `all` |
| `--phase-unit` | Phase unit (`rad`, `deg`) | `rad` |
| `--wrap-phase` | Wrap phase to ±π or ±180° | False |
| `--slice-frequency` | Slice at frequency (GHz) | - |
| `--slice-bias` | Slice at bias (mA) | - |
| `--device` | Device label in plots | Dataset name |
| `--show` / `--no-show` | Open browser preview directly | `--show` |
| `--save-html` / `--no-save-html` | Save HTML output | `--no-save-html` |
| `--output`, `-o` | HTML output path | auto-generated |

## Examples

**All views**
```bash
uv run sc plot flux-dependence LJPAL6572_B44D1
```

**Phase only (degrees)**
```bash
uv run sc plot flux-dependence --view phase --phase-unit deg LJPAL6572_B44D1
```

**Specific slices**
```bash
uv run sc plot flux-dependence \
    --slice-frequency 6.0 \
    --slice-bias 0.0 \
    LJPAL6572_B44D1
```

## Notes

!!! note "CLI architecture"
    This feature is now under the `sc plot` command tree (no standalone `flux-dependence-plot`).

!!! warning "Current data prerequisite"
    `sc plot flux-dependence` requires existing 2D `amplitude/phase` records in the database.
    `sc preprocess flux` raw TXT parsing is not fully implemented yet, so the command may show `Skip` if no suitable records exist.

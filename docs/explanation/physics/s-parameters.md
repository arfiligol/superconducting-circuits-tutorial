# S/Z/Y 參數

理解散射參數 (S)、阻抗參數 (Z)、導納參數 (Y) 是分析電路的基礎。

## S 參數（散射參數）

S 參數描述端口之間的功率波反射和傳輸。

### $S_{11}$：反射係數

$$S_{11} = \frac{b_1}{a_1}$$

- $|S_{11}| = 0$：完全匹配，無反射
- $|S_{11}| = 1$：完全反射

### 相位

$S_{11}$ 的相位在共振頻率處會發生 180° 跳變。

## Z 參數（阻抗參數）

$$V = Z \cdot I$$

阻抗矩陣描述電壓和電流的關係。

### 在 JosephsonCircuits.jl 中

```julia
sol = hbsolve(...; returnZ=true)
Z = sol.linearized.Z[1, :, 1, :, :]  # 提取 Z 矩陣
```

## Y 參數（導納參數）

$$I = Y \cdot V \quad \text{where} \quad Y = Z^{-1}$$

### 計算 Y11

```julia
Z_mat = sol.linearized.Z[1, :, 1, :, :]
Y_mat = inv.(Z_mat)
Y11 = Y_mat[1, 1, :]
```

導納的虛部 $\text{Im}(Y_{11})$ 在共振頻率處過零點。

## 轉換關係

| From | To S | To Z | To Y |
|------|------|------|------|
| S | - | $(1+S)(1-S)^{-1} Z_0$ | ... |
| Z | $(Z-Z_0)(Z+Z_0)^{-1}$ | - | $Z^{-1}$ |
| Y | ... | $Y^{-1}$ | - |

## 實用範例

尋找共振頻率的方法：

1. 找 $\angle S_{11}$ 相位跳變點
2. 找 $\text{Im}(Y_{11}) = 0$ 的零點

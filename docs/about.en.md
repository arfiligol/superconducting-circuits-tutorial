# About

This site teaches superconducting circuits from physics foundations to engineering practice, with an emphasis on transferable conceptual frameworks.

## What This Site Covers

- Shared superconducting-circuit physics backbone (Qubits and JPAs)
- Practical workflows from simulation to analysis
- Reference materials for CLI, data formats, and guardrails

## How to Use It

- New readers: start from `Documentation -> Tutorials`
- Physics-first readers: go to `Documentation -> Explanation`
- Tooling-first readers: go to `Documentation -> Reference`

## 電路圖繪製 (Circuit Diagrams)

我們使用 **Schemdraw** (Python) 來生成所有電路圖，確保風格一致且可維護。

### 規範要求
- **格式**: SVG (向量圖，可縮放)
- **原始碼**: 必須附上 Schemdraw 程式碼，方便日後修改

# plot-admittance

Plot preprocessed admittance ($Im(Y)$) data to inspect resonance extraction quality.

## Usage

```bash
uv run plot-admittance [OPTIONS] [components ...]
```

## Arguments

| Argument | Description | Default |
|---|---|---|
| `components` | Component IDs or JSON paths under `data/preprocessed/` | - |

## Options

| Option | Description | Default |
|---|---|---|
| `--mode` | Visualization mode (`lines`, `heatmap`, `both`) | `both` |
| `--show-zeros` | Mark zero crossings ($Im(Y)=0$) | False |
| `--freq-min` | Min frequency (GHz) | - |
| `--freq-max` | Max frequency (GHz) | - |
| `--title` | Plot title | - |
| `--matplotlib` | Use Matplotlib backend (default: Plotly) | False |

## Examples

**Lines + heatmap**
```bash
uv run plot-admittance LJPAL658_v1
```

**Mark zero crossings and limit range**
```bash
uv run plot-admittance --show-zeros --freq-min 4.0 --freq-max 8.0 LJPAL658_v1
```

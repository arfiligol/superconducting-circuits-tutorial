# flux-dependence-plot

Plot flux dependence scan data as heatmaps and slices.

## Usage

```bash
uv run flux-dependence-plot [OPTIONS] [components ...]
```

## Arguments

| Argument | Description | Default |
|---|---|---|
| `components` | Dataset names or IDs | Built-in defaults |

## Options

| Option | Description | Default |
|---|---|---|
| `--parameter` | Parameter name (e.g. `S11`) | `S11` |
| `--view` | View (`amplitude`, `phase`, `combined`, `all`) | `all` |
| `--phase-unit` | Phase unit (`rad`, `deg`) | `rad` |
| `--wrap-phase` | Wrap phase to ±π or ±180° | False |
| `--slice-frequency` | Slice at frequency (GHz) | - |
| `--slice-bias` | Slice at bias (mA) | - |
| `--device` | Device label in plots | Dataset name |
| `--matplotlib` | Use Matplotlib backend (default: Plotly) | False |

## Examples

**All views**
```bash
uv run flux-dependence-plot LJPAL6572_B44D1
```

**Phase only (degrees)**
```bash
uv run flux-dependence-plot --view phase --phase-unit deg LJPAL6572_B44D1
```

**Specific slices**
```bash
uv run flux-dependence-plot \
    --slice-frequency 6.0 \
    --slice-bias 0.0 \
    LJPAL6572_B44D1
```

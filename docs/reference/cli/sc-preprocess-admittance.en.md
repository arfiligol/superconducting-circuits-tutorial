# sc preprocess admittance

Import HFSS Admittance CSV files into SQLite Dataset (formerly `sc-preprocess-admittance`).

## Usage

```bash
uv run sc preprocess admittance [OPTIONS] [csv ...]
```

## Arguments

| Argument | Description | Default |
|----------|-------------|---------|
| `csv` | HFSS Admittance CSV path(s) or directory | (None) |

## Options

| Option | Description | Default |
|--------|-------------|---------|
| `--dataset-name` | Override Dataset Name (Single file only) | |
| `--tags` | Comma-separated tags | |

## Examples

**Basic Import**
```bash
uv run sc preprocess admittance data/raw/admittance/LJPAL658_v1_Im_Y11.csv
```

**Batch Import**
```bash
uv run sc preprocess admittance data/raw/admittance/
```

**Override Name**
```bash
uv run sc preprocess admittance \
    --dataset-name MyJPA_v2 \
    data/raw/admittance/raw_export_final.csv
```

## Notes

!!! note "Database import"
    The current pipeline imports into SQLite; JSON output is deprecated.

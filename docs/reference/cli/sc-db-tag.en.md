# sc-db tag

Manage Tags (create/update/delete supported).

## Usage

```bash
uv run sc-db tag <action> [args]
```

## Arguments

| Argument | Description | Default |
|---|---|---|
| `<action>` | `list` / `create` / `update` / `delete` / `auto-reorder` | - |

## Options

| Option | Description | Default |
|---|---|---|
| `--help` | Show help | - |

## Examples

**List**
```bash
uv run sc-db tag list
```

**Create**
```bash
uv run sc-db tag create "NewTag"
```

**Rename (sync links)**
```bash
uv run sc-db tag update "NewTag" "RenamedTag"
```

**Delete**
```bash
uv run sc-db tag delete "RenamedTag"
```

**Reorder ID**
```bash
uv run sc-db tag auto-reorder
```

## Notes

!!! warning "Link updates"
    `update` renames the tag and keeps all DatasetTagLink relations.

# sc-db data-record

Manage DataRecord (read/delete only).

## Usage

```bash
uv run sc-db data-record <action> [args]
```

## Arguments

| Argument | Description | Default |
|---|---|---|
| `<action>` | `list` / `info` / `delete` / `reorder-id` | - |

## Options

| Option | Description | Default |
|---|---|---|
| `--help` | Show help | - |

## Examples

**List**
```bash
uv run sc-db data-record list
```

**Info**
```bash
uv run sc-db data-record info <ID>
```

**Delete**
```bash
uv run sc-db data-record delete <ID>
```

**Reorder ID**
```bash
uv run sc-db data-record reorder-id <OLD_ID> <NEW_ID>
```

## Notes

!!! warning "Destructive"
    `delete` and `reorder-id` are destructive. Back up first.

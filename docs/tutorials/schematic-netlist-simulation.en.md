# From Netlist to Simulation

This page connects Schema and Simulation into a short execution flow.

## Flow

1. Author Source Form in Schema Editor and save.
2. Verify Expanded Netlist Configuration in Simulation.
3. Configure sources, solver options, and optional post-processing.
4. Run simulation and inspect Raw/Post-Processed Result Views.

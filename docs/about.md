# 關於本教學

這是一份從物理到工程的超導電路教學網站，聚焦在可以遷移到不同研究題目的核心框架。

## 網站涵蓋內容

- 超導電路物理主幹（Qubit 與 JPA 共用的核心概念）
- 從模擬到分析的實用工作流
- 包含 CLI、資料格式與專案 Guardrails 的參考資料

## 如何使用本網站

- 新讀者：建議從 `Documentation -> Tutorials (導引式流程)` 開始
- 物理背景讀者：可前往 `Documentation -> Explanation (物理主幹)`
- 工具/開發導向讀者：可前往 `Documentation -> Reference (參考中心)`

## 目前狀態

本頁面目前先放置簡介版內容，後續將補齊更完整的 Roadmap、受眾分層與學習路徑圖。

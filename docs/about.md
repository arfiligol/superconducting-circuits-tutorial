# About

這是一份從物理到工程的超導電路教學網站，聚焦在可以遷移到不同研究題目的核心框架。

## What This Site Covers

- Superconducting circuits physics backbone (Qubit and JPA shared concepts)
- Practical workflows from simulation to analysis
- Reference materials for CLI, data formats, and guardrails

## How to Use It

- New reader: start from `Documentation -> Tutorials`
- Physics-first reader: go to `Documentation -> Explanation`
- Tooling-first reader: go to `Documentation -> Reference`

## Current Status

這個頁面目前先放簡介版內容，後續會補齊更完整的 roadmap、受眾分層與學習路徑圖。
